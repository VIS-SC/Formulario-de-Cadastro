# Formulario de Cadastro

## Enunciado do exercício

Você deverá enviar um link do seu github com:

* Formulário de Cadastro de clientes contendo
- Nome
- E-mail
- CPF ou CNPJ
- Telefone
- CEP
- Logradouro
- Número
- Bairro
- Cidade
- Estado

* Listagem destes dados
* Exclusão de algum item da lista
